# <a name="dynamics-365-remote-assist-faq"></a>Dynamics 365 Remote Assist – Häufig gestellte Fragen

Hier sind Antworten auf häufige Fragen zur Verwendung von [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)].

Weitere Informationen finden Sie unter [Hilfe zu HoloLens](https://support.microsoft.com/products/hololens).

Wenn Sie ein IT-Administrator sind und [!include[pn-remote-assist](../includes/pn-remote-assist.md)] auf Geräten bereitstellen möchten, auf denen die [!include[pn-hololens](../includes/pn-hololens.md)] Commercial Suite ausgeführt wird, lesen Sie [Remote Assist kaufen und bereitstellen](buy-and-deploy-remote-assist.md). 

## <a name="can-i-try-the-app-before-buying-it"></a>Kann ich die App testen, bevor ich sie kaufe?

Ja, [!include[pn-remote-assist](../includes/pn-remote-assist.md)] ist als 90-Tage-Testversion verfügbar. Weitere Informationen finden Sie unter [Remote Assist kostenlos testen](try-remote-assist-free.md). Beachten Sie, dass die kostenlose 90-Tage-Testversion nur für Benutzer verfügbar ist, die noch kein Abonnement haben. 

## <a name="can-i-still-use-the-preview-app"></a>Kann ich die Vorschau-App verwenden?

Die Vorschau ist beendet. Vorhandene Kunden der Vorschau-App können bis März 2019 ein Upgrade auf ein kostenpflichtiges Abonnement durchführen. 

## <a name="what-are-the-technical-requirements-for-installing-remote-assist"></a>Was sind die technischen Anforderungen für die Installation von Remote Assist?

Zur Verwendung von [!include[pn-remote-assist](../includes/pn-remote-assist.md)] ist Folgendes erforderlich:

-   [Ein Abonnement für Dynamics 365 Remote Assist.](../licensing/buy-and-deploy.md)

-   Ein [HoloLens auf dem das Windows 10 April 2018 Update](https://support.microsoft.com/en-us/help/12643) ausgeführt wird. 

-   Um einen Videoanruf vorzunehmen, benötigen Sie einen Kontakt, der die aktuelle Version von [Microsoft Teams](https://products.office.com/microsoft-teams/group-chat-software) auf einem PC mit Windows 10 oder einem mobilen Gerät mit Microsoft Teams Mobile verwendet. Microsoft [!include[pn-teams](../includes/pn-teams.md)] ist [als kostenloser Download](https://teams.microsoft.com/downloads) verfügbar.

## <a name="do-i-need-a-subscription-to-microsoft-teams-or-office-365"></a>Benötige ich ein Abonnement von Microsoft Teams oder Office 365?

Seit dem 7. Januar 2019 ist ein Teams-Abonnement in dem Abonnement von Remote Assist für Einzelpersonen enthalten, die eine Remote Assist-Lizenz auf HoloLens verwenden. Eine Microsoft Teams-Lizenz ist auch für Personen (Experten) erforderlich, die mit einem Benutzer von Remote Assist auf HoloLens kommunizieren. Teams kann für diese Benutzer als [kostenloser Download unter](https://teams.microsoft.com/downloads) zur Verfügung stehen. 

## <a name="how-often-is-remote-assist-updated"></a>Wie häufig wird Remote Assist aktualisiert?

[!include[pn-remote-assist](../includes/pn-remote-assist.md)] entspricht der Modern Lifecyle-Richtlinie mit regelmäßigen Dienstupdates und größeren Updates alle 6 Monate (Oktober/April). 

## <a name="how-can-i-tell-what-version-im-using"></a>Wie finde ich heraus, welche Version ich verwende?

Rufen Sie **Einstellungen** auf und Sie sehen die Versionsnummer im Abschnitt **Info**.

## <a name="why-am-i-not-getting-incoming-call-notifications"></a>Warum werde ich nicht über eingehende Anrufe benachrichtigt?

Dies kann beispielsweise der Fall sein, wenn Sie sich bei [!include[pn-remote-assist](../includes/pn-remote-assist.md)] mit einem anderen Konto angemeldet haben als dem, das Sie für die Anmeldung bei Ihrem [!include[pn-hololens](../includes/pn-hololens.md)] verwendet haben. Achten Sie darauf, dass Sie sich bei der App und dem Gerät mit demselben Konto anmelden. 

## <a name="i-see-a-message-saying-i-have-no-internet-connection"></a>In einer Meldung wird angegeben, dass kein Zugang zum Internet möglich ist.

Zum Verwenden von [!include[pn-remote-assist](../includes/pn-remote-assist.md)] benötigen Sie eine WLAN-Verbindung. Sorgen Sie dafür, dass Ihr [!include[pn-hololens](../includes/pn-hololens.md)] [mit einem WLAN verbunden ist](https://support.microsoft.com/en-us/help/12629).

## <a name="remote-assist-is-frozen-or-unresponsive"></a>Remote Assist ist gesperrt oder reagiert nicht.

Schließen Sie [!include[pn-remote-assist](../includes/pn-remote-assist.md)] und öffnen Sie es neu. Wenn es weiter Probleme gibt, [starten Sie HoloLens](https://support.microsoft.com/en-us/help/12642) neu.

## <a name="remote-assist-isnt-showing-my-contacts"></a>Remote Assist zeigt meine Kontakte nicht an.

Wenn Sie die erwarteten Kontakte nicht sehen, versuchen Sie Folgendes:

-   Sie müssen sich mindestens einmal bei [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-teams](../includes/pn-teams.md)] mit dem Konto angemeldet haben, das Sie in Ihrem [!include[pn-hololens](../includes/pn-hololens.md)] verwenden.

-   Schließen Sie [!include[pn-remote-assist](../includes/pn-remote-assist.md)] und öffnen Sie es neu.

-   [Starten Sie Ihr HoloLens](https://support.microsoft.com/en-us/help/12642) neu.

## <a name="the-mixed-reality-toolbar-doesnt-appear-on-the-pc-teams-client-during-a-remote-assist-call"></a>Die Mixed Reality-Symbolleiste wird während eines Remote Assist-Anrufs nicht auf dem PC Teams-Client angezeigt.

Dies kann passieren, wenn [!include[pn-teams](../includes/pn-teams.md)] eine neue Version des Client im Hintergrund heruntergeladen hat, aber weiterhin die ältere Version der App verwendet. Durch einen Neustart des PCs wird dieses Problem normalerweise behoben.

## <a name="why-does-audio-sometimes-cut-out-even-in-quiet-environments"></a>Warum wird der Ton teilweise auch in ruhigen Umgebungen abgeschaltet?

Wenn [!include[pn-remote-assist](../includes/pn-remote-assist.md)] in einer lauten Umgebung verwendet wird, sollten an beiden Enden Kopfhörer verwendet werden, um die Kommunikation zu verbessern. Wenn Kopfhörer für beide Benutzer verwendet werden, kann Echofilterung aktiv Geräuschunterdrückung aktiviert, wodurch der Ton abgeschaltet wird.

## <a name="can-i-use-remote-assist-to-place-a-call-between-two-separate-tenants"></a>Kann ich mit Remote Assist einen Anruf zwischen zwei separaten Mandanten vornehmen?

Ja, aber nur, wenn der Experte als Gast vom Mandanten des anderen Benutzers aus anruft.
Derzeit bietet [!include[pn-remote-assist](../includes/pn-remote-assist.md)] keine Unterstützung für zwei Benutzer, die sich gegenseitig anrufen, auch wenn zwei Mandanten verbunden sind.

## <a name="does-microsoft-teams-include-the-same-remote-assist-features-on-all-platforms-and-devices"></a>Enthält Microsoft Teams auf allen Plattformen und Geräten dieselben Remote Assist-Funktionen?

Nein. Freihand, Pfeile und Hinzufügen von Dateien ist nur auf [!include[pn-teams](../includes/pn-teams.md)] für PC verfügbar.

## <a name="i-signed-in-to-my-hololens-but-i-still-cant-sign-in-to-remote-assist"></a>Ich habe mich bei meinem HoloLens angemeldet, kann mich aber dennoch nicht bei Remote Assist anmelden.

Wenn Sie Schwierigkeiten haben, sich bei [!include[pn-remote-assist](../includes/pn-remote-assist.md)] anzumelden, versuchen Sie Folgendes:

-   Verwenden Sie unbedingt ein Geschäfts-, Schul- oder Unikonto, kein [!include[cc-microsoft](../includes/cc-microsoft.md)]-Konto für die Anmeldung bei [!include[pn-hololens](../includes/pn-hololens.md)].

-   Stellen Sie sicher, dass das Arbeits- oder Schulkonto, das Sie verwenden, über ein gültiges Abonnement für [Office 365 Business](https://products.office.com/en-us/business/office) Premium oder Essentials mit aktiviertem [Microsoft Teams](https://products.office.com/en-us/microsoft-teams/group-chat-software) verfügt.

-   Sorgen Sie dafür, dass Ihr [!include[pn-hololens](../includes/pn-hololens.md)] mit dem Internet verbunden ist.

-   Fragen Sie bei Ihrem Administrator nach, ob Ihr Gerät und Netzwerk für die Arbeit mit [!include[pn-remote-assist](../includes/pn-remote-assist.md)] eingerichtet sind.

## <a name="how-do-i-set-up-dynamics-365-for-field-service-to-work-with-remote-assist"></a>Wie richte ich Dynamics 365 for Field Service für die Funktion mit Remote Assist ein?

Informationen zum Einrichten der [!include[pn-field-service](../includes/pn-field-service.md)]-Integration mit [!include[pn-remote-assist](../includes/pn-remote-assist.md)], sodass ein Mitarbeiter mit direktem Kundenkontakt einen Experten zu einem [!include[pn-field-service](../includes/pn-field-service.md)]-Arbeitsauftrag anrufen kann, finden Sie unter [Problembehandlung für die Field Service-Integration mit Remote Assist](troubleshoot-field-service.md).

## <a name="what-languages-is-remote-assist-available-in"></a>In welchen Sprachen ist Remote Assist erhältlich?

[!include[pn-remote-assist](../includes/pn-remote-assist.md)] (nur in englischer Sprache verfügbar).

## <a name="where-is-remote-assist-available"></a>Wo ist Remote Assist verfügbar?

[!include[pn-remote-assist](../includes/pn-remote-assist.md)], das [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-hololens](../includes/pn-hololens.md)] erfordert, ist weltweit verfügbar.

[!include[pn-hololens](../includes/pn-hololens.md)]-Geräte sind für die folgenden Märkte verfügbar (nur auf Englisch):

Australien, Belgien, Bulgarien, China, Kanada, Kroatien, Dänemark, Deutschland, Estland, Finnland, Frankreich, Griechenland, Hong Kong, Island, Irland, Italien, Japan, Lettland, Liechtenstein, Litauen, Niederlande, Neuseeland, Norwegen, Österreich, Polen, Portugal, Rumänien, Singapur, Slowakei, Slowenien, Spanien, Schweden, Schweiz, Tschechische Republik, Türkei, Ungarn, Vereinigtes Königreich, Vereinigte Staaten, Zypern

## <a name="is-group-calling-available-in-remote-assist"></a>Sind Gruppenanrufe in Remote Assist verfügbar?

Ja, Gruppenanrufe sind ab dem April-Release von Remote Assist verfügbar. Bis zu den Personen 50 können an einem Videoanruf beteiligt sein.

## <a name="what-file-formats-does-the-insert-file-feature-support"></a>Welche Dateiformate werden durch die Funktion zum Dateieinfügen unterstützt?

Die Funktion zum Dateieinfügen unterstützt die meisten Bildformate (JPG, GIF, PNG) und PDF-Dateien.

## <a name="can-i-share-files-from-sharepoint-on-remote-assist"></a>Kann ich Dateien von SharePoint auf Remote Assist freigeben?

Nein, Sie können nur lokale Dateien und Dateien aus [!include[pn-onedrive](../includes/pn-onedrive.md)] freigeben.
