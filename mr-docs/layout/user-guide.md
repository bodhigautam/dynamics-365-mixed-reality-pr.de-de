# <a name="dynamics-365-layout-user-guide"></a>Dynamics 365 Layout-Benutzerhandbuch

Verwenden Sie [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-dyn-365-layout](../includes/pn-dyn-365-layout.md)], um Raumpläne auf Microsoft [!include[pn-hololens](../includes/pn-hololens.md)] oder auf einem immersiven [!include[pn-ms-windows-short](../includes/pn-ms-windows-short.md)] Mixed Reality-Headset zu entwerfen und auszugestalten. Importieren Sie 3D-Modelle und entwerfen, bearbeiten und arbeiten Sie virtuell zusammen – im realen Maßstab.

Benötigen Sie weitere Hilfe? [Überprüfen Sie die Häufig gestellten Fragen zu Dynamics 365 Layout](faq.md) für Antworten zu allgemeinen Fragen.

[Sehen Sie sich die Anleitungsvideos zu Dynamics 365 Layout an.](videos.md)

## <a name="what-youll-need"></a>Was Sie benötigen

-   [Ein Abonnement von Dynamics 365 Layout.](../licensing/buy-and-deploy.md) Das [!include[pn-layout](../includes/pn-layout.md)]-Abonnement enthält auch [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-dyn-365-import-tool](../includes/pn-dyn-365-import-tool.md)] für PCs.
    
    > [!TIP]
    > Sie können [!include[pn-layout](../includes/pn-layout.md)] bis zu 90 Tage lang kostenlos ausprobieren. [Erfahren Sie mehr über die kostenlose 90-Tage-Testversion von Dynamics 365 Layout](try-layout-free.md).

-   Ein [HoloLens](https://www.microsoft.com/hololens)-Gerät mit dem [Windows 10 April 2018 Update](https://support.microsoft.com/en-us/help/12643) und/oder ein immersives [Windows Mixed Reality-Headset](https://www.microsoft.com/en-us/windows/windows-mixed-reality). 

-   Ein [Windows Mixed Reality–ready PC](https://www.microsoft.com/en-us/windows/windows-mixed-reality-devices#wmrpcs) mit dem [Windows 10 April 2018 Update](https://support.microsoft.com/en-us/help/4028685).

-   Eine Internetverbindung.

## <a name="get-started-with-dynamics-365-layout-take-advantage-of-learning-tools"></a>Erste Schritte mit Dynamics 365 Layout: Nutzen Sie die Vorteile von Lerntools

[!include[pn-layout](../includes/pn-layout.md)] enthält zwei Werkzeuge (Teaching Moments und Demo-Layouts), mit denen Sie lernen können, wie Sie allgemeine Aufgaben wie Messen, Kopieren, Gruppieren, Ändern der Größe und Zugriff auf Einstellungen durchführen können.

### <a name="teaching-moments"></a>Teaching Moments

Teaching Moments bietet eine Reihe von Bildern, die es leicht machen, gängige Aufgaben zu verstehen. Um Teaching Moments zu aktivieren, tippen Sie auf **Hilfe** und anschließend auf den Bereich, für den Sie Hilfe benötigen.

![Teaching Moments](media/teaching-moments.PNG "Teaching Moments")

### <a name="demo-layouts"></a>Demo-Layouts

Demo-Layouts verwendet vordefinierte Layouts, um zu zeigen, wie verschiedene Teile einer Produktionsstätte erstellt werden. Sie können aus drei Demos wählen:
- Kopieren/Einfügen und Kollision
- Gruppieren und messen
- Anpassen und ändern der Größe

![Demo-Layouts](media/demo-layouts.PNG "Demo-Layouts")

Lernen Sie beispielsweise in der Demo „Kopieren/Einfügen und Kollision”, wie ein Förderband durch Kopieren und Positionieren von Abschnitten vervollständigt wird.

Um Demo-Layouts zu aktivieren, wählen Sie das gewünschte Demo aus dem **Layouts**-Menü aus.

![Demo-Layouts aktivieren](media/access-demo-layouts.PNG "Demo-Layouts aktivieren")

## <a name="use-dynamics-365-layout"></a>Verwenden von Dynamics 365 Layout

Um [!include[pn-layout](../includes/pn-layout.md)] zu verwenden, erstellen Sie auf Ihrem [!include[pn-hololens](../includes/pn-hololens.md)] oder PC Raumpläne und verwenden anschließend ein immersives [!include[pn-hololens](../includes/pn-hololens.md)] oder [!include[pn-ms-windows-short](../includes/pn-ms-windows-short.md)] Mixed Reality-Headset, um 3D-Modelle in den Layouts zu platzieren. Die [!include[pn-layout](../includes/pn-layout.md)] App beinhaltet einen Satz vorinstallierter Modelle. Sie können aber auch eigene hinzufügen.

### <a name="create-a-layout"></a>Erstellen eines Layouts

Beim Erstellen eines Layouts gibt es zwei Möglichkeiten:

-   Verwenden Sie [Microsoft Visio](https://products.office.com/en-us/visio/), um einen Grundriss zu entwerfen, und exportieren Sie diesen in [!include[pn-hololens](../includes/pn-hololens.md)] bzw. [!include[pn-ms-windows-short](../includes/pn-ms-windows-short.md)] Mixed Reality.

-   Verwenden Sie [!include[pn-layout](../includes/pn-layout.md)] auf dem [!include[pn-hololens](../includes/pn-hololens.md)], um einen Raum zu scannen.

#### <a name="create-a-layout-with-microsoft-visio"></a>Erstellen eines Layouts mit Microsoft Visio

**Schritt 1: Laden Sie das [!include[pn-visio](../includes/pn-visio.md)]-Add-In für [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-dyn-365-layout](../includes/pn-dyn-365-layout.md)]** herunter und installieren Sie es

1.  Öffnen Sie [!include[pn-import-tool](../includes/pn-import-tool.md)].

2.  Wählen Sie  **Über** aus und anschließend den 32- oder den 64-Bit-Download (wählen Sie denjenigen aus, der zu Ihrer [!include[pn-visio](../includes/pn-visio.md)]-Version passt).
    
    > [!div class="mx-imgBorder"]
    > ![Visio-Add-In herunterladen](media/visio-download.PNG "Visio-Add-In herunterladen") 

3.  Öffnen Sie [!include[pn-visio](../includes/pn-visio.md)] und wählen Sie dann **Datei** \> **Optionen** \> **Add-Ins**.

4.  Wählen Sie **COM Add-Ins** und anschließend **Weiter** neben **Verwalten**.

5.  Wählen Sie das **[!include[pn-visio](../includes/pn-visio.md)]-Add-In für [!include[pn-dyn-365-layout](../includes/pn-dyn-365-layout.md)]** und anschließend **Hinzufügen** aus.

**Schritt 2: Erstellen Sie Ihren Grundriss mit [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-visio](../includes/pn-visio.md)]**

-   Wenn der Grundriss bereit ist, wählen Sie die Registerkarte **[!include[pn-layout](../includes/pn-layout.md)]** aus, wählen Sie **Exportieren** aus und speichern Sie anschließend den Grundriss.

**Schritt 3: Importieren Sie den Grundriss mithilfe des [!include[pn-import-tool](../includes/pn-import-tool.md)]**

1.  Wählen Sie im [!include[pn-import-tool](../includes/pn-import-tool.md)] **Dieser PC** \> **Layout hinzufügen** und navigieren Sie anschließend zum Grundriss.

2.  Wählen Sie den Grundriss und anschließend **Öffnen** aus.

    Der Grundriss wird automatisch in [!include[pn-layout](../includes/pn-layout.md)] auf einem immersivem Mixed Reality-Headset, das mit demselben PC verbunden ist, angezeigt.

Um das Layout auf [!include[pn-hololens](../includes/pn-hololens.md)]zu verwenden, müssen Sie es dorthin verschieben:

1.  Verbinden Sie Ihren [!include[pn-hololens](../includes/pn-hololens.md)] mit Ihrem PC mithilfe des USB-Kabels.

2.  Öffnen Sie [!include[pn-import-tool](../includes/pn-import-tool.md)] auf dem PC und wählen Sie anschließend **Dieser PC** \> **Layouts** aus.

3.  Aktivieren Sie das Kontrollkästchen der Layouts, die Sie kopieren möchten, und wählen Sie dann **Senden** aus.
    
    > [!div class="mx-imgBorder"]
    > ![Layout zu HoloLens verschieben](media/selected-layouts-hololens.PNG "Layout zu HoloLens verschieben")

4.  Öffnen Sie die [!include[pn-layout](../includes/pn-layout.md)]-App auf [!include[pn-hololens](../includes/pn-hololens.md)]. Sie finden Ihr Layout auf der Registerkarte **Layouts**.

#### <a name="create-a-layout-with-hololens"></a>Erstellen eines Layouts mit HoloLens

1.  Öffnen Sie auf Ihrem [!include[pn-hololens](../includes/pn-hololens.md)]-Gerät die [!include[pn-layout](../includes/pn-layout.md)]-App.

2.  Wählen Sie **Neues Layout** aus und befolgen Sie anschließend die Anweisungen zum Scannen des Raumes und zur Erstellung eines Layouts.

Zum Verschieben des Layouts auf Ihren PC, um es auf einem immersiven Mixed Reality-Headset zu öffnen: 

1.  Verbinden Sie den [!include[pn-hololens](../includes/pn-hololens.md)] mit dem PC mithilfe des USB-Kabels.

2.  Öffnen Sie [!include[pn-import-tool](../includes/pn-import-tool.md)] und wählen Sie dann den [!include[pn-hololens](../includes/pn-hololens.md)]-Namen aus.

3.  Aktivieren Sie das Kontrollkästchen der Layouts, die Sie an Ihren PC senden möchten, und wählen Sie dann **Senden** aus.
    
    > [!div class="mx-imgBorder"]
    > ![Layout von HoloLens verschieben](media/selected-layouts-pc.PNG "Layout von HoloLens verschieben")


## <a name="place-copy-resize-rotate-and-select-3d-models"></a>Platzieren, Kopieren, Ändern der Größe, Rotieren und Auswählen von 3D-Modellen

In [!include[pn-hololens](../includes/pn-hololens.md)] und in [!include[pn-ms-windows-short](../includes/pn-ms-windows-short.md)] Mixed Reality können Sie 3D-Modelle platzieren, verschieben, kopieren, rotieren und ihre Größe ändern.

### <a name="on-hololens"></a>In HoloLens

| **Ziel**       | **Vorgehensweise**                                                                                                                                                                                                              |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Platzieren        | Wählen Sie **Modelle** aus, tippen und visieren Sie anschließend das Modell an, das Sie platzieren möchten. Verwenden Sie Anvisieren, um das Modell zu verschieben, und tippen Sie erneut, um es zu platzieren.                                                                      |
| Verschieben         | Visieren Sie ein Modell an. Tippen und halten Sie es. Bewegen Sie Ihre Hand, um das Modell neu zu positionieren. Geben Sie es anschließend frei.                                                                                                                         |
| Kopieren         | Tippen und visieren Sie das Modell an, um es auszuwählen. Wählen Sie dann **Kopieren**. Verschieben Sie das Modell an die gewünschte Stelle und tippen Sie dann, um es zu platzieren. Wählen Sie **Fertig**.                                                                         |
| Ändern der Größe       | Tippen und visieren Sie das Modell an, um es auszuwählen. Wählen Sie dann **Größe ändern**. Tippen und halten Sie eine beliebige Kante des Modells, und bewegen Sie anschließend Ihre Hand hinein oder heraus.                                                                                   |
| Rotieren       | Tippen und visieren Sie das Modell an, um es auszuwählen. Wählen Sie einen der Rotationspfeile aus, um 90 Grad zu drehen. Für eine präzisere Rotation, tippen und halten Sie das runde Rotationssteuerelement und bewegen Ihre Hand anschließend für die Drehung.                     |
| Mehrfachauswahl | Visieren Sie eine leere Stelle an. Tippen und halten Sie. Bewegen Sie anschließend Ihre Hand diagonal, um ein Rechteck über alle Modelle zu bilden, die Sie auswählen möchten. Visieren Sie eines der ausgewählten Modelle an, um zu sehen, was Sie mit allen Modellen machen können. |

### <a name="on-an-immersive-headset"></a>**Auf einem immersiven Headset**

| **Ziel**       | **Vorgehensweise**                                                                                                                                                                                                                                                                          |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Platzieren        | Wählen Sie **Modelle** im App-Menü aus. Richten Sie Ihren Controller auf das gewünschte Modell. Drücken und halten Sie anschließend den Trigger, um es aufzunehmen. Zeigen Sie auf die Stelle, an der das Modell platziert werden soll, und lassen Sie dann den Trigger los.                                                                        |
| Verschieben         | Zeigen Sie auf das Modell. Halten Sie den Trigger gedrückt. Bewegen Sie Ihren Controller und lassen anschließend den Trigger zum Platzieren los.                                                                                                                                                                  |
| Kopieren         | Wählen Sie das Modell aus, und wählen Sie dann **Kopieren**. Zeigen Sie mit Ihrem Controller auf die Stelle, an der Sie die Kopie platzieren möchten. Drücken Sie den Trigger erneut, um sie zu platzieren.Um mehrere Kopien zu platzieren, zeigen und wählen Sie erneut. Sie müssen nicht zum Original zurückkehren. Wählen Sie **Fertig** nach der letzten Kopie aus.      |
| Ändern der Größe       | Wählen Sie das Modell aus und anschließend **Größe ändern**. Zeigen Sie auf eine Kante des Objekts. Halten Sie den Trigger gedrückt. Ziehen Sie die Kante und geben Sie dann den Trigger frei.                                                                                                                      |
| Rotieren       | Zeigen Sie auf das gewünschte Modell und halten Sie den Trigger gedrückt. Zeigen Sie anschließend auf einen der Rotationspfeile und drücken Sie den Trigger, um das Modell um 90 Grad zu drehen. Für eine präzisere Rotation verwenden Sie das runde Rotationssteuerelement: Zeigen Sie auf das Modell, ziehen und halten Sie den Trigger und bewegen Sie nun den Controller zum Drehen. |
| Mehrfachauswahl | Zeigen Sie mit Ihrem Controller auf eine leere Stelle. Ziehen Sie halten Sie den Trigger, um über die gewünschten Modelle ein Rechteck zu ziehen. Zeigen Sie auf eins der ausgewählten Modelle, um zu sehen, was Sie damit machen können. |

## <a name="lock-a-layout"></a>Sperren eines Layouts

Sie können ein Layout sperren, um unbeabsichtigte Änderungen daran zu verhindern. Dies ist besonders dann hilfreich, wenn Sie ein Layout zu Überprüfungszwecken für Ihre Kollegen freigeben möchten. 

Zum Sperren eines Layouts tippen Sie über **Layout sperren** im Randleistenmenü in die Luft. Dadurch wird das aktuelle Layout für die Bearbeitung gesperrt.

![Layout sperren](media/locked-layout.PNG "Layout sperren")

In der oberen rechten Ecke des Layout-Titels wird ein Sperrsymbol angezeigt.
 
![Symbol für gesperrtes Layout](media/locked-layout-graphic.PNG "Symbol für gesperrtes Layout")
 
Wenn ein Layout gesperrt ist und eine Person ein Modell im Layout auswählt, wird nach dem dritten aufeinanderfolgenden Versuch ein Popup angezeigt, um mitzuteilen, dass das Layout für die Bearbeitung gesperrt ist. Eine ähnliches Popup wird angezeigt, wenn versucht wird, ein Modell aus dem **Layout**-Menü im aktuellen Layout zu platzieren.

Zum Entsperren eines Layouts tippen Sie über **Layout entsperren** im Randleistenmenü in die Luft. 
 
![Layout entsperren](media/unlock-layout.PNG "Layout entsperren")

## <a name="improve-performance-when-your-models-dont-require-texture-or-multiple-colors"></a>Verbessern der Leistung, wenn Ihre Modelle keine Textur oder mehrere Farben benötigen

In bestimmten Szenarien sind Texturen oder mehrere Farben zur Visualisierung von 3D-Modellen möglicherweise nicht wichtig. In diesen Fällen sollten Sie zur Verbesserung der Leistung Texturen deaktivieren und eine einzige Farbe verwenden. Verwenden Sie dazu die Einstellung **Leistung**. 

Zum Aktivieren oder Deaktivieren der Einstellung **Leistung**:

- Wählen Sie **Einstellungen**, **Leistung** und anschließend **Vereinfachen von Objekten durch Verwendung von nur einer Farbe** aus.

> [!div class="mx-imgBorder"]
> ![Leistungsmodus-Einstellung](media/performance-mode-setting.PNG "Leistungsmodus-Einstellung")

## <a name="turn-off-fading-of-holograms"></a>Ausschalten des Verblassens von Hologrammen

Standardmäßig verblassen Hologramme, wenn Sie sich innerhalb von 45 Zentimetern zu ihnen befinden. Sie können die Einstellung **Hologramm verblassen** verwenden, wenn Sie diesen Effekt beenden möchten und nah an Ihre Hologramme heran möchten.

![Hologramm verblassen-Einstellung](media/hologram-fade.PNG "Hologramm verblassen-Einstellung")

Schalten Sie „Hologramm verblassen” jederzeit wieder ein, indem Sie die Einstellung auf **An** zurücksetzen.
